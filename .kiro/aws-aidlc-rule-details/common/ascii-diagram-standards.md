# ASCII图表标准

## 必须：仅使用基本ASCII

**关键**：始终使用基本ASCII字符绘制图表（最大兼容性）。

### ✅ 允许：`+` `-` `|` `^` `v` `<` `>` 和字母数字文本

### ❌ 禁止：Unicode框线字符
- 不要使用：`┌` `─` `│` `└` `┐` `┘` `├` `┤` `┬` `┴` `┼` `▼` `▲` `►` `◄`
- 原因：在不同字体/平台上渲染不一致

## 标准ASCII图表模式

### 关键：字符宽度规则
**框中的每一行必须具有完全相同的字符数（包括空格）**

✅ 正确（所有行 = 67个字符）：
```
+---------------------------------------------------------------+
|                      组件名称                                 |
|  这里是描述文本                                               |
+---------------------------------------------------------------+
```

❌ 错误（宽度不一致）：
```
+---------------------------------------------------------------+
|                      组件名称                                 |
|  这里是描述文本                                          |
+---------------------------------------------------------------+
```

### 框模式
```
+-----------------------------------------------------+
|                                                     |
|              计算器应用程序                         |
|                                                     |
|  通过基于Web的界面为用户提供基本算术运算            |
|                                                     |
+-----------------------------------------------------+
```

### 嵌套框
```
+-------------------------------------------------------+
|              Web服务器（PHP运行时）                   |
|  +-------------------------------------------------+  |
|  |  index.php（单体应用程序）                      |  |
|  |  +-------------------------------------------+  |  |
|  |  |  HTML模板（视图层）                       |  |  |
|  |  |  - 表单渲染                               |  |  |
|  |  |  - 结果显示                               |  |  |
|  |  +-------------------------------------------+  |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

### 箭头和连接
```
+----------+
|   源     |
+----------+
     |
     | HTTP POST
     v
+----------+
|   目标   |
+----------+
```

### 水平流
```
+-------+     +-------+     +-------+
| 步骤1 | --> | 步骤2 | --> | 步骤3 |
+-------+     +-------+     +-------+
```

```
用户操作流程：
    |
    v
+----------+
|   输入   |
+----------+
    |
    | 验证
    v
+----------+
|   处理   |
+----------+
    |
    | 返回
    v
+----------+
|   输出   |
+----------+
```

## 验证

创建图表前：
- [ ] 仅基本ASCII：`+` `-` `|` `^` `v` `<` `>`
- [ ] 无Unicode框线
- [ ] 使用空格（不是制表符）对齐
- [ ] 角落使用 `+`
- [ ] **所有框线相同字符宽度**（计算包括空格的字符）
- [ ] 测试：在等宽字体中验证角落垂直对齐
